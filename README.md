# 环境监控与设备控制终端

## 项目简介
基于STM32F103C8T6的裸机综合项目，覆盖9个外设（GPIO、UART、定时器、ADC、DMA、I2C、PWM、中断、看门狗），目标是构建一个完整的嵌入式系统，夯实外设的使用，学会架构，搭配

## 当前进度
- [x] 阶段一：系统骨架搭建，LED心跳验证
- [ ] 阶段二：逐个点亮外设（串口）
- [ ] 阶段三：系统集成与状态机联调

## 硬件平台
- 主控：STM32F103C8T6
- 传感器：BMP280 (I2C)
- 显示：SSD1306 OLED (I2C)
- 交互：RGB灯、蜂鸣器、按键、串口
